## :wrench: Setup
- Requirements: Python >= 3.5 `python3 -V`
- Install dependencies on a virtualenv `python3 -m pip install --user virtualenv`:

```sh
python3 -m virtualenv env
source env/bin/activate
pip install -r requirements.txt
```

## :rocket: Start Server
```sh
bash start.sh
```
